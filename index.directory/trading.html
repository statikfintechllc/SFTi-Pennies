Connect to IBKR

Authenticate with Interactive Brokers to access live trading data

Note: This feature requires the IBKR Client Portal Gateway to be running locally or on your server. Authentication will redirect you to IBKR's login page, then back to this application.

  1. Ensure IBKR Client Portal Gateway is running
  2. Click "Connect to IBKR" to start OAuth flow
  3. Login with your IBKR credentials
  4. Authorize the application
  5. You'll be redirected back with your session token

By connecting, you agree to IBKR's terms of service and API usage policies
