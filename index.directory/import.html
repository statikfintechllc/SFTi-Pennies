Expected CSV Format (Robinhood)

Your CSV file should contain the following columns: 

Symbol, Activity Date, Activity Type, Quantity, Price, Amount
AAPL, 2025-01-15, Buy, 10, 150.00, -1500.00
AAPL, 2025-01-16, Sell, 10, 155.00, 1550.00

Required Columns:

  • Symbol - Stock ticker symbol
  • Activity Date - Trade date (YYYY-MM-DD)
  • Activity Type - Buy or Sell
  • Quantity - Number of shares
  • Price - Price per share
  • Amount - Total transaction amount

Optional Columns:

  • Strategy - Trading strategy used
  • Tags - Comma-separated tags (e.g., "scalp,momentum")
  • Notes - Trade notes or comments
